Bagaimana Talos Bekerja?

Setiap pesan masuk diproses melalui dual-pass intent system: classifier regex cepat diikuti oleh decider AI dengan confidence scoring.

Berdasarkan intent, orchestrator memilih pipeline yang tepat dan menjalankan rantai role agents (researcher → planner → architect → builder → checker → verifier).

Setiap step divalidasi. Jika gagal, sistem melakukan auto-retry hingga 2x sebelum fallback. Semua output disimpan ke memory engine untuk pembelajaran kontekstual.

Hasilnya? Agen yang semakin pintar setiap kali digunakan.

🚀 Getting Started

Mulai dalam 3 Langkah

1

Clone Repository

Download atau clone proyek Talos dari GitHub.

git clone https://github.com/SoonOver/AIPRANK.git
2

Setup Environment

Copy file .env.example, isi API key Gemini, dan install dependencies.

npm install && cp .env.example .env
3

Jalankan Talos

Aktifkan agen dan mulai berinteraksi melalui terminal.

npm start
