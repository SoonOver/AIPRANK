# HelloWorld

Aplikasi sederhana yang menampilkan teks "Hello, World!" ke layar.

## Cara Install

1. Buat folder baru dengan nama "HelloWorld".
2. Buat file baru dengan nama "HelloWorld.java" di dalam folder "HelloWorld".
3. Tulis kode Java di dalam file "HelloWorld.java" seperti di atas.
4. Jalankan aplikasi dengan menggunakan perintah `javac HelloWorld.java` untuk mengompilasi kode Java menjadi kode bytecode, kemudian jalankan aplikasi dengan perintah `java HelloWorld`.

## Cara Jalankan

1. Jalankan perintah `javac HelloWorld.java` untuk mengompilasi kode Java menjadi kode bytecode.
2. Jalankan aplikasi dengan perintah `java HelloWorld`.

## Fitur

* Menampilkan teks "Hello, World!" ke layar.
